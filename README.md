# PMS.Rent Blog

This repository hosts my personal blog built with GitHub Pages.  
The website contains articles, guides, and resources related to short‑term rental management, automation, multilingual communication, and property management workflows.

## About the Project
The blog is generated using GitHub Pages and Markdown files.  
All articles are stored in the repository and automatically published at:

https://sisikiev.github.io

## Structure
- `index.md` — main page of the blog  
- `_posts/` — folder containing blog articles in Markdown format  
- `assets/` — images and static files (optional)  
- `_config.yml` — configuration file for GitHub Pages and Jekyll (optional)

## Purpose
The goal of this blog is to share knowledge, best practices, and insights for property managers in Europe and Ukraine, helping them improve operations and guest experience using modern tools like PMS.Rent.
